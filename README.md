# improvedApriori

Improved Apriori method is an enhanced approach that refines the classical Apriori method to effectively reveal potential associations between drugs/vaccines and AEs from post-marketing safety monitoring datasets, especially when AEs are rare.

## Code folders
1. `improvedApriori` R package
2. Simulation code files
3. VAERS data Apriori implementation
4. Kuo et al dataset implementation

## Data folders
1. Simulated data
   1. Setting 1: one drug and three AEs
   2. Setting 2: three drugs and five AEs
   3. Setting 3: five drugs and ten AEs
3. VAERS data
